# Security Policy

## Supported Versions

Please use only the latest release versions of Scanline Classic for
production. We don't have the resources to maintain older versions.

| Version | Supported          |
| ------- | ------------------ |
| Latest  | :white_check_mark: |
| <Latest | :x:                |

## Reporting a Vulnerability

Create an issue on GitHub. In addition, we encourage you to publish a
report about it on some other public forum of your choosing. We will
update the issue within 30 days. If we decline to fix the
vulnerability, a reason will be provided. You are welcome to reopen
the issue by presenting new information that makes the original
reasoning invalid.
